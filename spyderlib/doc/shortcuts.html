Global shortcutsÂ¶

The default Spyder global shortcuts are the following:

  • Maximizing current plugin / restoring original window layout:

    Ctrl+Shift+Alt+M

  • Go to Editor:

    Ctrl+Shift+E

  • Go to Find in files:

    Ctrl+Shift+F

  • Go to Interactive Console:

    Ctrl+Shift+C

  • Go to External Console:

    Ctrl+Shift+X

  • Go to Workspace:

    Ctrl+Shift+W

  • Go to History log:

    Ctrl+Shift+H

  • Go to Object inspector:

    Ctrl+Shift+I

  • Go to Online help:

    Ctrl+Shift+D
