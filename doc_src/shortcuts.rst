Global shortcuts
================

The default Spyder global shortcuts are the following:

- Maximizing current plugin / restoring original window layout:

  Ctrl+Shift+Alt+M

- Go to :doc:`editor`:
  
  Ctrl+Shift+E
  
- Go to :doc:`findinfiles`:
  
  Ctrl+Shift+F
  
- Go to :doc:`console`:

  Ctrl+Shift+C
  
- Go to :doc:`extconsole`:

  Ctrl+Shift+X
  
- Go to Workspace:

  Ctrl+Shift+W
  
- Go to :doc:`historylog`:

  Ctrl+Shift+H
  
- Go to :doc:`inspector`:

  Ctrl+Shift+I
  
- Go to :doc:`onlinehelp`:

  Ctrl+Shift+D
